Λωρίδες που προορίζονται αποκλειστικά για μέσα μαζικών μεταφορών

Μπορούν επίσης να χρησιμοποιούνται από ταξί και οχήματα προτεραιότητας Τα άλλα οχήματα μπορούν να χρησιμοποιούν τις λωρίδες αποκλειστικής κυκλοφορίας μόνον όταν το επιτρέπουν τα σήματα της τροχαίας / οδικά σήματα, για να περάσουν δίπλα από σταθμευμένα οχήματα ή άλλα εμπόδια, να προσπεράσουν, να στρίψουν (και να κάνουν στροφή επιτόπου) ή να αποκτήσουν πρόσβαση σε άλλο δρόμο.

Λωρίδες για ποδηλάτες

Τα οχήματα προτεραιότητας μπορούν να χρησιμοποιούν τις λωρίδες για ποδηλάτες. Τα άλλα οχήματα μπορούν να χρησιμοποιούν τις λωρίδες για ποδηλάτες μόνον όταν το επιτρέπουν τα σήματα της τροχαίας / οδικά σήματα, για να περάσουν δίπλα από σταθμευμένα οχήματα ή άλλα εμπόδια, να προσπεράσουν, να στρίψουν (και να κάνουν στροφή επιτόπου) ή να αποκτήσουν πρόσβαση σε άλλο δρόμο.
