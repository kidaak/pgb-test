Οι λωρίδες που προορίζονται αποκλειστικά για τα μέσα μαζικής μεταφοράς και τα ταξί φέρουν ειδική σήμανση γι΄αυτό το σκοπό.
Τα ποδήλατα επιτρέπονται στις λωρίδες αποκλειστικής κυκλοφορίας αν αυτό ορίζεται στα σήματα της τροχαίας.

Άλλα οχήματα μπορούν να χρησιμοποιούν τις λωρίδες αποκλειστικής κυκλοφορίας (εκτός αν αυτό συνεπάγεται υπέρβαση συνεχούς γραμμής) όταν πρόκειται:

  • όταν ετοιμάζονται να στρίψουν
  • να πάρουν ή να αφήσουν επιβάτες (εκτός αν αυτό εμποδίζει τη ροή τακτικών δημόσιων συγκοινωνιών)
