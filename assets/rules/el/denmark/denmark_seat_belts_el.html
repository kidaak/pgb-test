Η χρήση ζωνών ασφαλείας και παιδικών καθισμάτων είναι ΥΠΟΧΡΕΩΤΙΚΗ.

Δεν υποχρεούνται να φορούν ζώνες ασφαλείας οι εξής:

  • οι οδηγοί που κάνουν όπισθεν ή οδηγούν σε χώρους στάθμευσης ή παρόμοιους χώρους
  • οι οδηγοί και επιβάτες με αναπηρία
  • άτομα που εξαιρούνται για ιατρικούς λόγους
  • οι οδηγοί που μεταφέρουν πελάτη
  • οι οδηγοί και επιβάτες οχημάτων προτεραιότητας, όπως τα ασθενοφόρα
  • τα οχήματα παράδοσης ή τα ιδιωτικά αυτοκίνητα που κάνουν συχνές στάσεις (κάθε 500 μ. ή λιγότερο)
  • οι αστυνομικοί και άλλα άτομα ειδικών υπηρεσιών που μεταφέρουν κρατουμένους
  • οι οδηγοί ενόπλων δυνάμεων όταν οδηγούν με χαμηλή ταχύτητα σε περιοχές ασκήσεων
  • όσοι πραγματοποιούν τακτικές υπηρεσίες μεταφορών κατά τις οποίες επιτρέπονται όρθιοι επιβάτες

Τα παιδικά καθίσματα δεν είναι υποχρεωτικά:

  • για παιδιά κάτω των 3 ετών που ταξιδεύουν στο πίσω κάθισμα ταξί.
  • για παιδιά άνω των 3 ετών που ταξιδεύουν στο πίσω κάθισμα ταξί φορώντας ζώνη ασφαλείας
  • για παιδιά άνω των 3 ετών που ταξιδεύουν στο πίσω κάθισμα αυτοκινήτου περιστασιακά
  • σε οχήματα όπου δεν υπάρχει χώρος για τρίτο παιδικό κάθισμα και το παιδί ταξιδεύει στο πίσω κάθισμα φορώντας ζώνη ασφαλείας
