Στην Ιρλανδία η οδήγηση γίνεται στα αριστερά.

Αυτοκινητόδρομοι

  • Οι οδηγοί πρέπει να οδηγούν μόνον εμπρός. Δεν επιτρέπεται να στρίψουν ή να κάνουν όπισθεν.
  • Οι οδηγοί πρέπει να οδηγούν με τέτοια ταχύτητα και κατά τέτοιο τρόπο, ώστε να μην παρακωλύουν την κυκλοφορία στον αυτοκινητόδρομο.
  • Οι οδηγοί απαγορεύεται να οδηγούν σε τμήματα του αυτοκινητόδρομου που δεν είναι λωρίδες κυκλοφορίας, π.χ. λωρίδες έκτακτης ανάγκης, παρά μόνον εάν πράγματι πρόκειται για έκτακτη ανάγκη.
  • Οι οδηγοί δεν πρέπει να σταματούν ή να σταθμεύουν σε κανένα τμήμα του αυτοκινητόδρομου, εκτός εάν το όχημά τους έχει βλάβη ή τους το έχει ζητήσει η αστυνομία (Garda).
  • Στη λωρίδα κυκλοφορίας που βρίσκεται πιο κοντά στον κεντρικό διάμεσο του αυτοκινητόδρομου (την εξωτερική λωρίδα ή λωρίδα προσπέρασης), οι οδηγοί απαγορεύεται να οδηγούν οχήματα που δεν επιτρέπεται να υπερβαίνουν τα 90 km/h, εκτός εάν το γενικό όριο ταχύτητας δεν υπερβαίνει τα 90km/h.
  • Οι οδηγοί απαγορεύεται να σταματούν για αποβίβαση ή επιβίβαση επιβατών στον αυτοκινητόδρομο.
  • Οι οδηγοί οφείλουν να οδηγούν στην αριστερή λωρίδα του αυτοκινητόδρομου εκτός εάν προσπερνούν άλλο όχημα.

Κυκλικοί κόμβοι

Οι οδηγοί είναι υποχρεωμένοι από τον νόμο να εισέρχονται στους κυκλικούς κόμβους στρίβοντας αριστερά. Πρέπει να συμπεριφέρονται στους κυκλικούς κόμβους όπως σε μια διασταύρωση και να δίνουν προτεραιότητα στα οχήματα που ήδη βρίσκονται εντός του κόμβου (που έρχονται από τα δεξιά). Εάν ένας οδηγός δεν είναι εξοικειωμένος με την οδήγηση στα αριστερά του δρόμου, οι κυκλικοί κόμβοι μπορεί να αποδειχθούν ιδιαίτερα επικίνδυνοι (οι οδηγοί θα πρέπει να είναι ιδιαίτερα προσεκτικοί).
